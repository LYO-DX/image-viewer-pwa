\# Image Viewer PWA



\## 日本語



このプロジェクトは、Android向けに最適化されたシンプルな \*\*画像ビューア PWA（Progressive Web App）\*\* です。  

特徴は以下の通りです：

(テスト中です)

\- 特定の画像を表示し、ピンチズーム・ドラッグで拡大縮小・位置調整可能

\- 画像を回転させることが可能

\- 初期画像とユーザーが選択した画像の切り替えが可能

\- 全画面表示中は画面固定、1タップで下部に薄い解除ボタンを表示（2秒で自動消失）

\- 軽量でシンプルなUIデザイン

\## English



This project is a simple \*\*Image Viewer PWA (Progressive Web App)\*\* optimized for Android.  

Key features:

(This is a test)

\- Display a specific image with pinch-zoom and drag support for resizing and repositioning

\- Rotate the displayed image

\- Switch between the initial image and a user-selected image

\- Fullscreen mode locks the screen; a tap shows a thin exit button at the bottom (disappears automatically after 2 seconds)

\- Lightweight and simple UI design
